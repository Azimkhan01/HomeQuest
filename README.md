# HomeQuest

## User Roles and Permissions

### 1. User

#### What can a user do, and how can they use HomeQuest?

- **Signup & Login:**

  - Users need to sign up using a legitimate email address to receive important notifications.
  - Every login attempt is recorded, and users are informed of their login time.

- **Home Page:**

  - Upon successful login with the correct email and password, users are redirected to the home page.

- **Property Management:**

  - Users can add properties for listing.
  - Users can edit, view, and delete their listings from their profile page.

- **Appointments & Agent Interaction:**

  - Users can schedule appointments with agents.
  - Users may allocate slots for their listings to other users.

- **Profile & Settings:**

  - Users can reset their password.
  - Users can update their profile information.
  - Profile page includes management of listings and interactions.

- **Feed & Updates:**

  - Users can view agent-posted feeds, which contain property updates and listings.

This structured guide ensures clarity and ease of understanding for new users and contributors.
